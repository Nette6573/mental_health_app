# Mental Health AI Assistant (Jamaica)

This project contains:

- A fine-tuned GPT-2 model for mental-health conversational support
- Backend API (FastAPI)
- Frontend (web + mobile)
- Jamaican health facility recommendation system
- Bible verse + character guidance

## Folders
- backend/ → AI API + health info routes
- training/ → Model training + testing
- data/ → Training datasets + health facility JSON files
